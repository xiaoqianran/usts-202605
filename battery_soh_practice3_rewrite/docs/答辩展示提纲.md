# 实践三答辩展示提纲

## 第 1 页：任务与数据

- 任务：基于 NASA 锂离子电池老化数据预测 SOH。
- 数据：B0005、B0006、B0007、B0018 四个电池的 discharge cycle。
- 标签：SOH(%) = 当前容量 / 额定容量 2.0Ah * 100。
- 插图：`outputs/report_assets/01_capacity_degradation.png`。

## 第 2 页：数据预处理与划分

- 预处理：解析 mat 文件，保留 discharge cycle，过滤容量异常、时间异常和非有限值。
- A：单电池随机 60%/20%/20%。
- B：单电池按 cycle 前 60% 训练/验证，后 40% 测试。
- C：源电池 + 目标前 10% 训练，目标后 90% 测试。
- 插图：`outputs/report_assets/02_split_schematic.png`。

## 第 3 页：特征选择

- 特征类型：电压统计、温度统计、电流统计、放电时长、到达电压阈值时间。
- 方法：只在训练集计算 PCC，按绝对值排序，选择 Top K。
- 防止泄露：capacity_ah、charge_ah_integral、energy_wh 不进入模型输入。
- 插图：`outputs/report_assets/03_pcc_heatmap_topK.png`。
- 表格：`outputs/08_topK_features_and_reasons.csv`。

## 第 4 页：模型设计

- 模型：MLP 回归网络。
- 结构：K -> 64 -> 32 -> 1。
- 优化：AdamW，SmoothL1Loss，验证集早停。
- 选择理由：输入是结构化 cycle 特征，样本量较小，MLP 简洁稳定且便于解释。
- 插图：`outputs/report_assets/04_mlp_structure.png`。

## 第 5 页：训练测试结果

- 表格：`outputs/05_metrics_summary_12runs.csv` 或 `outputs/06_metrics_by_case_mean.csv`。
- 插图：`outputs/report_assets/05_metrics_comparison_12runs.png`。
- 重点说明：A 是随机插值型实验，通常指标最高；B 更接近未来 cycle 外推；C 最能体现跨电池泛化难度。

- 预测曲线：`outputs/report_assets/09_prediction_true_vs_pred_B0005_B.png`。
- Loss 曲线：`outputs/report_assets/10_loss_curve_B0005_B.png`。

## 第 7 页：问题与解决

- 原始 mat 数据结构复杂：封装统一读取和 discharge cycle 抽取。
- 特征可能泄露标签：容量及由容量直接积分得到的特征只用于分析，不输入模型。
- 随机划分结果容易过高：增加时序外推和跨电池迁移实验，避免只展示简单场景。
- 小样本训练不稳定：固定随机种子、标准化输入输出、验证集早停。
