# 实践二答辩展示提纲

## 1. 实验目标

- 任务：在 MSTAR 目标分类数据集上验证标签数量不足对深度网络性能的影响。
- 对照组：全标签监督、10% 标签监督、10% 标签 + 剩余无标签 FixMatch。
- 核心观察：标签减少导致精度下降，半监督学习通过无标签样本的一致性约束恢复部分精度。

## 2. 方法说明

- 分类网络：SmallResNet，输入为 1 通道 SAR 灰度图，输出 10 类目标 logits。
- 有限标签划分：按类别分层采样，每类保留 `floor(n * 0.1)` 个有标签样本，保证类别均衡。
- FixMatch：弱增强图像生成伪标签，置信度超过阈值后监督强增强图像预测，训练目标为 `L = Lx + lambda_u Lu`。

## 4. 结果表述模板

全标签监督学习作为参考上限；当训练标签减少到 10% 以下时，监督模型测试精度明显下降，说明深度模型对标签数量敏感。FixMatch 在相同有标签样本基础上额外使用剩余无标签样本，通过伪标签和一致性正则提升测试精度，证明无标签数据能够缓解有限标签问题。

如果实际运行结果提升不明显，可说明原因：伪标签阈值过高会降低无标签样本利用率，阈值过低会引入错误伪标签；SAR 图像的强增强需要保守，否则可能破坏目标散射结构。
